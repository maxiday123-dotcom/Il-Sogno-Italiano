# Pre-Entrega Front End

Este proyecto lo armé como práctica para la cursada de Front End.  
La idea es mostrar lo básico de HTML y CSS aplicado en una página sencilla, con estilos externos, Google Fonts y fondos personalizados.

Incluye:
- Header y footer con estilos básicos.
- Sección de productos hecha con Flexbox.
- Sección de reseñas organizada con Grid.
- Sección de contacto responsiva con Media Queries.

El propósito es cumplir con la consigna de la pre-entrega y tener una base lista para seguir mejorando en próximos trabajos.
